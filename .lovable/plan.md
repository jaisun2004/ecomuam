

# Propagate Comparison Date Range to All Charts via Context

## Overview

Create a global DateRange context so every chart can detect when comparison mode is active and render a second data series (dashed line/bar) representing the comparison period. All mock data generators will produce both primary and comparison values.

## Changes

### 1. New file: `src/contexts/DateRangeContext.tsx`

Create context with:
- `currentRange`, `compareRange`, `compareEnabled`, `timePreset`
- Setters for each
- `useDateRange()` hook

### 2. Refactor `src/components/Topbar.tsx`

Remove local state for `timeRange`, `compareEnabled`, `currentRange`, `compareRange`. Consume from `useDateRange()` context instead.

### 3. Wrap app: `src/pages/Index.tsx`

Add `<DateRangeProvider>` wrapping `<IndexInner />` inside the existing provider stack.

### 4. Update all 16 view files with charts

For each chart across all views, when `compareEnabled` is true:

- **Line charts**: Add a second `<Line>` with `strokeDasharray="5 5"` and a muted color, using a `_compare` data key
- **Area charts**: Add a second `<Area>` with reduced opacity and dashed stroke
- **Bar charts**: Add a second `<Bar>` with reduced opacity side-by-side
- **Composed charts**: Add comparison series matching the primary series type

Each view will:
1. Import `useDateRange`
2. Generate comparison mock data (offset values by ~10-15%) alongside primary data
3. Conditionally render comparison series when `compareEnabled` is true
4. Add a small legend indicator showing "Comparison period" with dashed styling

### Comparison visual pattern

```text
Primary:    ────────────  (solid, full opacity)
Comparison: - - - - - -   (dashed, 40% opacity, lighter color)
```

A small legend row below each chart: `── Current period  --- Comparison period`

### Technical details
- Comparison data generated by applying a random offset (-15% to +10%) to primary mock data
- `useMemo` wraps comparison data generation to prevent re-renders
- No new dependencies needed — uses existing Recharts, existing Calendar/Popover components
